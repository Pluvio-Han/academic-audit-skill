# Formatting and Submission

## General Checks

- Title reflects the actual contribution and method.
- Abstract states problem, method, evidence/results, and conclusion.
- Keywords are specific and searchable.
- Headings are hierarchical and consistent.
- Figures and tables are referenced in text.
- Captions are self-contained.
- Equations, figures, tables, sections, and appendices cross-reference correctly.
- Bibliography style matches target venue.
- Page limit, word count, margin, font, line spacing, anonymization, and supplement rules are respected.

## LaTeX Checks

- No unresolved `??` references.
- No citation placeholders like `[?]`.
- `booktabs` tables avoid vertical rules unless template requires them.
- Math notation is defined before use.
- Labels are unique and descriptive.
- Figures use vector formats when appropriate.
- Long tables and algorithms do not overflow margins.

## DOCX Checks

- Styles are used consistently for headings, captions, references, and body text.
- Automatic numbering is intact.
- Figure/table captions are not plain manual text when the document needs generated lists.
- References are consistently formatted.

## Required Statements

Require each statement only when the condition applies:

| Statement | Require When |
|---|---|
| Data availability | Empirical work uses datasets not fully included in the paper, or target venue requests it |
| Code availability | Computational, ML, simulation, or algorithmic work where results depend on implementation |
| Ethics approval / informed consent | Human subjects, patient data, interviews, surveys, private records, or sensitive personal data |
| Funding | Journal/conference submission or project report where funding affected the work |
| Conflict of interest | Journal/conference submission, policy analysis, industry-funded research, or grant proposal |
| Author contributions | Multi-author formal submission where CRediT or author-role disclosure is required |
| Acknowledgments | External support, data access, mentorship, compute resources, or institutional help not listed as funding |
| AI tool use disclosure | Target venue or competition explicitly requires it; otherwise flag as optional, not blocking |
| Limitations | All serious academic submissions; absence is at least `MAJOR` unless the format forbids it |

Do not require all statements universally; require them when the field, venue, or content makes them relevant.

## Chinese Academic and Competition Formatting

When auditing Chinese or mixed Chinese-English manuscripts:

- Check whether the target requires GB/T 7714 citation style, Chinese abstract, English abstract, Chinese keywords, or English keywords.
- Verify Chinese punctuation consistency and avoid mixing full-width and half-width punctuation in the same context.
- Check numbering style for sections, figures, tables, equations, and appendices against the template.
- For 全国大学生数学建模竞赛 and similar competitions, prioritize title page, abstract, keywords, problem restatement, assumptions, model construction, solution, sensitivity analysis, strengths/weaknesses, references, and appendices.
- Keep technical terms consistent across Chinese and English. If a term is introduced bilingually, use the same Chinese translation throughout.
- Do not impose GB/T 7714 or Chinese layout rules unless the venue, institution, or competition requires them.
