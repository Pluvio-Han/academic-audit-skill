# Contributing

Contributions are welcome if they improve the skill's practical audit quality.

Good contributions usually do one of the following:

- Add a concise, field-specific audit rule.
- Improve competition-paper scoring relevance.
- Make an audit output more evidence-bound and actionable.
- Improve Chinese/English academic style checks.
- Add missing submission or formatting checks.

Please keep the skill compact. Prefer adding detailed material to `academic-audit/references/` instead of expanding `SKILL.md` unless the instruction is core to every audit.

Avoid adding broad writing-generation behavior. This skill is primarily for auditing manuscripts, not drafting them.
